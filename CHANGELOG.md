# Change Log

All notable changes to the "vscode-diff-html-exporter" extension will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/) and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.0.1] - 2026-02-27

### Added

- Initial release of Diff HTML Exporter
- Export diff editor content as HTML file
- Side-by-side diff view in exported HTML
- File list navigation in exported HTML
- Command accessible from:
  - Editor title bar (export icon)
  - Editor context menu
  - Command Palette
- Multi-language support:
  - English (en)
  - Japanese (ja)
- Support for diff editor comparisons
- Auto-generated filename based on compared files
- Standalone HTML output with CDN-based styling
