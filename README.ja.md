# Diff HTML Exporter

[English](README.md) | 日本語

Visual Studio Codeの差分エディタの内容を、HTMLファイルとしてエクスポートできる拡張機能です。

## 機能

- **差分をHTMLへエクスポート**: 差分エディタでの比較内容をスタンドアロンのHTMLファイルに変換
- **サイドバイサイド表示**: エクスポートされるHTMLは、読みやすいサイドバイサイドの比較ビューを含みます
- **スタンドアロン出力**: エクスポートされたHTMLファイルは、CDN経由で必要なスタイルをすべて含んでいます
- **多言語対応**: インターフェースは英語と日本語で利用可能です

## 使い方

1. VS Codeの差分エディタで2つのファイルを開く（比較モード）
2. エディタのタイトルバーにあるエクスポートアイコンをクリック、または
3. エディタ上で右クリックして「Diff HTML Exporter: 比較結果をHTMLに出力」を選択、または
4. コマンドパレット（macOSでは`Cmd+Shift+P`、Windows/Linuxでは`Ctrl+Shift+P`）を開き、「Diff HTML Exporter: 比較結果をHTMLに出力」を検索
5. エクスポートするHTMLファイルの場所とファイル名を選択
6. 生成されたHTMLファイルをWebブラウザで開いて差分を確認

## 必要要件

- Visual Studio Code バージョン 1.109.0 以上

## 拡張機能の設定

この拡張機能は、VS Codeの設定項目を追加しません。
